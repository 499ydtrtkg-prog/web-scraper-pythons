# Python Web Scraper

A Python script that automatically scrapes data from websites and exports clean CSV files.

## What it does
- Scrapes multiple pages automatically
- Extracts structured data (title, price, rating)
- Exports to CSV ready for Excel or Google Sheets

## Libraries used
- requests
- BeautifulSoup4

## How to run
pip3 install requests beautifulsoup4
python3 scraper.py
